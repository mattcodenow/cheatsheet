# Cascading Stylesheets (CSS)
[<< Back To Home](../README.md)

## Basics

### Syntax
```
selector {
  attribute: value;
  attribute2: value2;
}
```

### External Document
Adding an external .css file to an .html file:
```html
<!-- inside the html head tag -->
<head>
  <link rel="stylesheet" type="text/css" href="./path/to/styles.css" />
</head>
```
